
## Compilation

- sudo apt install qt5-qmake qt5-default # in ubuntu
- mkdir ~/stage; cd ~/stage
- git clone https://github.com/plops/cl-gen-qt-thing
- cd cl-gen-qt-thing/; mkdir build
- cd build; cmake ../source -DCMAKE_BUILD_TYPE=RELEASE; make
